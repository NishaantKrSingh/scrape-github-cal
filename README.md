## Usage
Make a virtual environment for Python
```bash
python3 -m venv venv
```

Activate the virtual environment
```bash
source venv/bin/activate
```

Install the required packages
```bash
pip install -r requirements.txt
```

Replace user name in the code if needed
```python
# Username
usr = "your_name"
```

Run code
```bash
python3 main.py
```
